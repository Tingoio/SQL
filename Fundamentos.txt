1-Banco de dados é um conjunto de fato (dados) que formam informações que se transformar em conhecimento, um sistema de gerenciamento de dados funcional é essencial para uma gestão eficiente

2-Dado é um fato sem relação nenhuma com algum objeto ou situação específica, informação é o dado em um contexto específico, conhecimento é a informação com alguma função

3-O sistema de arquivos normalmente é feito manualmente e é mais complexo de ser organizado do que um SGBD(Sistema Gerenciador de Banco de Dados)
